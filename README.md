# Weather Diary (Дневник погоды)

## Автор
**Лобканов Кирилл**

## Описание программы
Weather Diary - это GUI-приложение для ведения дневника погоды. Позволяет записывать, просматривать и анализировать погодные условия. Данные сохраняются в формате JSON.

### Основные возможности:
- 📝 Добавление записей о погоде (дата, температура, описание, осадки)
- 🔍 Фильтрация записей по дате и температуре
- 📊 Отображение статистики (средняя, мин, макс температура)
- 💾 Сохранение и загрузка данных в JSON формате
- 📁 Экспорт данных в текстовый файл
- 🎨 Цветовая индикация температуры (холодно/тепло/жарко)
- ✅ Валидация вводимых данных
